# 组件使用

## UI 组件库

项目使用 **Ant Design Vue 4.2** 作为 UI 组件库。

## Vben 组件

项目基于 Vben5，提供了一些封装组件：

- **VbenForm** - 表单组件
- **VbenTable** - 表格组件（基于 VXE Table）
- **VbenModal** - 模态框组件
- **VbenDrawer** - 抽屉组件

## 组件文档

详细的组件使用文档请参考 Vben5 官方文档。
