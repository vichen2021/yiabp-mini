## 实体 1：Category（图书分类）

### 基本信息

| 项目 | 值 |
|------|-----|
| 实体名 | Category |
| 实体中文名 | 图书分类 |
| 是否树形 | 是（有 ParentId 字段） |

### 业务字段

| 字段名 | 字段类型 | 字段中文名 | 说明 |
|--------|----------|------------|------|
| Name | string | 分类名称 | 必填，最大100字符 |
| Code | string | 分类编码 | 必填，唯一索引，如 "001-002" |
| ParentId | Guid | 父级ID | 树形结构，根节点为 null |
| Level | int | 层级 | 第几级分类，1=一级 |
| Status | CategoryStatusEnum | 分类状态 | 枚举：正常/停用 |

### 枚举定义

**枚举名**：CategoryStatusEnum

| 枚举值 | 值 | 说明 |
|--------|-----|------|
| Normal | 0 | 正常 |
| Disabled | 1 | 停用 |

### 默认字段（自动包含）

| 字段 | 类型 | 说明 |
|------|------|------|
| State | bool | 状态（默认 true） |
| OrderNum | int | 排序号（默认 0） |
| Remark | string? | 备注（可选，最大500字符） |
| IsDeleted | bool | 软删除标记 |
| CreationTime | DateTime | 创建时间 |
| CreatorId | Guid? | 创建者ID |
| LastModifierId | Guid? | 最后修改者ID |
| LastModificationTime | DateTime? | 最后修改时间 |

## 实体 2：Book（图书）

### 基本信息

| 项目 | 值 |
|------|-----|
| 实体名 | Book |
| 实体中文名 | 图书 |
| 是否树形 | 否 |

### 业务字段

| 字段名 | 字段类型 | 字段中文名 | 说明 |
|--------|----------|------------|------|
| Title | string | 书名 | 必填，最大200字符，索引字段 |
| ISBN | string | ISBN编号 | 必填，唯一索引，如 "978-7-111-12345-6" |
| Author | string | 作者 | 最大100字符 |
| Publisher | string | 出版社 | 最大100字符 |
| CategoryId | Guid | 分类ID | 关联 Category 表 |
| Price | decimal | 价格 | 图书定价，精度2位小数 |
| StockCount | int | 库存数量 | 当前库存数量 |
| BookStatus | BookStatusEnum | 图书状态 | 枚举：可借阅/已借出/损坏 |

### 枚举定义

**枚举名**：BookStatusEnum

| 枚举值 | 值 | 说明 |
|--------|-----|------|
| Available | 0 | 可借阅 |
| Borrowed | 1 | 已借出 |
| Damaged | 2 | 已损坏 |

### 默认字段（自动包含）

同 Category 实体。

## 实体 3：BorrowRecord（借阅记录）

### 基本信息

| 项目 | 值 |
|------|-----|
| 实体名 | BorrowRecord |
| 实体中文名 | 借阅记录 |
| 是否树形 | 否 |

### 业务字段

| 字段名 | 字段类型 | 字段中文名 | 说明 |
|--------|----------|------------|------|
| BookId | Guid | 图书ID | 关联 Book 表 |

| BorrowDate | DateTime | 借阅日期 | 借阅时间 |
| DueDate | DateTime | 应还日期 | 计算得出（借阅日期+借阅天数） |

**枚举名**：BorrowStatusEnum

| 枚举值 | 值 | 说明 |
|--------|-----|------|
| Borrowing | 0 | 借阅中 |
| Returned | 1 | 已归还 |
| Overdue | 2 | 已逾期 |

| 实体 | id (Guid) | 字符串类型 | 数字类型 | 枚举类型 |

| Category | ✅ Id | ✅ Name, Code | ✅ Level | ✅ Status |

## 生成命令

### 命令 1：创建模块

```

```

### 命令 2：生成 Category CRUD（树形实体）

```
/crud-generator-plus book-manage 模块，Category 实体（图书分类），树形结构。
字段：分类名称、分类编码、层级int、分类状态枚举（正常=0、停用=1）

```

## 执行顺序建议

1. **先创建模块** → `/module-generator book-manage`
2. **先生成 Category**（树形实体，作为 Book 的关联选项）
3. **再生成 Book**（CategoryId 可从 Category 下拉选择）
4. **最后生成 BorrowRecord**（BookId 可从 Book 下拉选择）
